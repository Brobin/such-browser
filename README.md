##Such Browser

A simple web browser written in Python

Dependencies
- gtk
- webkit

```
sudo apt-get install python-gtk2
sudo apt-get install python-webkit
```

To run
- download
- open a terminal in that directory
- change the permissions `chmod +x doge.py`
- run it `./doge.py`
